# 11. Components

> Authored: 2026-04-25 (Task 25)

This section documents the 13 UI components that form the Zuga component layer. Each component page covers anatomy, variants, accessibility (keyboard, ARIA, contrast, focus), code examples in Astro + Vue + plain HTML, a Don'ts table, and a full token reference.

All component tokens are Tier 3 (scoped, `--component-*`), compose from Tier 2 semantic tokens, and are listed in `packages/design-tokens/src/components/`.

## Component index

| Component | Summary |
|---|---|
| [Avatar](./11-components/avatar.md) | Circular identity element representing a user or entity |
| [Badge](./11-components/badge.md) | Small label for status, count, or categorization |
| [Button](./11-components/button.md) | Primary interactive trigger for actions |
| [Card](./11-components/card.md) | Surface container for grouping related content or actions |
| [Checkbox](./11-components/checkbox.md) | Binary selection control for forms and settings |
| [Input](./11-components/input.md) | Single-line text entry field |
| [Modal](./11-components/modal.md) | Overlay dialog for focused content or decisions |
| [Nav](./11-components/nav.md) | Navigation component for studio-level wayfinding |
| [Select](./11-components/select.md) | Dropdown for selecting a single option from a set |
| [Skeleton](./11-components/skeleton.md) | Animated loading placeholder |
| [Tabs](./11-components/tabs.md) | Segmented panel switcher within a view |
| [Toast](./11-components/toast.md) | Ephemeral notification for confirmations and alerts |
| [Toolbar](./11-components/toolbar.md) | Horizontal strip of context-sensitive controls |

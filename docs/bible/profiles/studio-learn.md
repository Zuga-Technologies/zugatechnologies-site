# studio-learn Profile

## Persona

ZugaLearn users are engaging with structured educational content — lessons, quizzes, skill tracks, and progress milestones. The mental state is receptive and patient; sessions are longer than a quick habit check-in and require sustained focus. The persona spans from early learners onboarding to new studio features to advanced users working through technical skill paths. The UI should feel structured and trustworthy — more textbook than app store.

## Accent rationale

ZugaLearn is the education family and uses indigo. Indigo is conventionally associated with knowledge, academia, and deep focus — closer to a library than a dashboard. It is sufficiently distinct from ZugaCode's blue (dev tools, IDE convention) and ZugaCloud's sky (infrastructure openness) to avoid confusion, while still reading as a serious, focused hue. See `acc14` in §5.

## Deviations from §15 whitelist

No deviations. This profile overrides only `--accent-*` (indigo ramp).

## Density posture

Comfortable density. No override.

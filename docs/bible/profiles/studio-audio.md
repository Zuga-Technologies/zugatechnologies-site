# studio-audio Profile

## Persona

ZugaAudio users are working with music and sound — generating audio, managing audio assets, and composing AI-assisted music tracks. The mental state is immersive and sensory; the visual UI competes with the sound itself for attention, so it should be bold enough to orient without distracting from the listening experience. Sessions can be long and exploratory, punctuated by short bursts of decision-making when a generated clip lands.

## Accent rationale

ZugaAudio is the music family and uses magenta. Magenta is distinct from the visual creative family's violet — both are saturated and expressive, but magenta carries a sonic/electric connotation (speaker grilles, waveform visualizers, stage lighting) that violet lacks. This keeps ZugaAudio from being visually swallowed by the creative cluster while still reading as a creative, energy-forward studio. See `acc11` in §5.

No deviations. This profile overrides only `--accent-*` (magenta ramp).

# studio-forge Profile

## Persona

ZugaForge users are building things inside the Zuga platform — registering creations, publishing studio plugins, and managing the forge's creation registry. The persona is closer to a platform developer or power user than a casual consumer: they've graduated from using studios to extending them. Sessions are intentional and goal-oriented; the user arrives knowing what they want to publish or register.

## Accent rationale

ZugaForge is the forge family and uses orange. Orange references the forge metaphor directly — heat, craft, material transformation. It is the only high-temperature, warm accent in the palette; every other studio uses cool or neutral hues. This makes ZugaForge immediately recognizable as the "build and publish" layer of the platform. See `acc12` in §5.

## Deviations from §15 whitelist

No deviations. This profile overrides only `--accent-*` (orange ramp).

## Density posture

Comfortable density. No override.

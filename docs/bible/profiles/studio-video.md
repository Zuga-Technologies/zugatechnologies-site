# studio-video Profile

## Persona

ZugaVideo users are generating or managing video content — AI-generated clips, dynamic desktop wallpapers sourced from provider APIs, and template-based video compositions. The session pattern is more deliberate than ZugaImage: a video takes time to generate, so users queue work and return. The UI needs a timeline-oriented layout that communicates job status and progress clearly, not just a gallery grid.

## Accent rationale

ZugaVideo is in the creative family and uses violet. It is a direct sibling of ZugaImage — both deal in AI-generated visual media — but the product surface is fundamentally different: ZugaVideo's primary workspace is a timeline/queue, not a gallery picker. The shared violet ramp communicates "same creative family" to returning users without the two studios competing visually. See `acc07` in §5.

## Deviations from §15 whitelist

No deviations. This profile overrides only `--accent-*` (violet ramp).

## Density posture

Comfortable density. No override.

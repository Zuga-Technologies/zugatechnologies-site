# studio-themes Profile

## Persona

ZugaThemes users are browsing, purchasing, and applying visual themes across all Zuga studios. The marketplace surface is unique: it must render every sub-brand accent correctly in theme preview cards without the marketplace's own accent colliding with the previewed theme. The user's goal is to change how the product looks, so the marketplace UI itself needs to be maximally neutral — a white-walled gallery for the themes to perform in.

## Accent rationale

ZugaThemes inherits master cyan — it earns no override. A marketplace that applies to all studios must not assert its own accent color, because any override would compete with the themed content it surfaces. Master cyan is the zero-opinion brand anchor. This is a deliberate "defer to master" decision, not a placeholder. See `acc13` in §5.

## Deviations from §15 whitelist

No deviations. This profile overrides only `--accent-*` — and in this case, not even that; it consumes Tier 2 defaults as-is.

## Density posture

Comfortable density. No override.

# studio-code Profile

## Persona

ZugaCode users are developers working inside their editor while an AI agent runs tasks in the background — reviewing code, executing tool calls, writing to files. The mental state is focused and task-oriented: they've delegated work to the system and expect clear status feedback. The studio surfaces agent activity, review queues, and hook output. Users are technical and dense; they prefer information over whitespace, but the layout is comfortable — not the data-wall of ZugaTrader.

## Accent rationale

ZugaCode is the dev family and uses blue. Blue is the canonical developer accent across the industry — IDEs, terminals, documentation sites. It reads "this is a technical tool" without ambiguity. Sky is taken by ZugaCloud (infrastructure, not dev-productivity), so blue is the right separation. See `acc09` in §5.

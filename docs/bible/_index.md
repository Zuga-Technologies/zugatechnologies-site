# Zuga Design Bible

Overview of the bible structure. The bible is organized into 18 sections covering mission, brand, visual foundations, components, patterns, accessibility, tokens, governance, and more.

## Sections

1. Mission & Principles
2. Brand Foundation
3. Voice & Tone
4. Color Foundations
5. Color in Practice
6. Typography
7. Spacing & Layout
8. Iconography
9. Imagery & Illustration
10. Motion
11. Components (see `11-components/`)
12. Patterns (see `12-patterns/`)
13. Accessibility
14. Code Tokens
15. Sub-brand Contract
16. Cross-platform
17. Governance
18. Glossary

Sub-directories:
- `11-components/` — 13 component reference pages
- `12-patterns/` — 7 pattern pages
- `adr/` — Architecture Decision Records
- `profiles/` — 21 sub-brand profile definitions
